# Legacy Prototype Code

This directory contains the original Ciphergy prototype files (pre-Phase 0).
These files are preserved for reference but are NOT part of the active codebase.

## Contents

| Directory / File | Description |
|---|---|
| `agents/` | Original agent specification markdown files |
| `ciphergy/` | Prototype Python package (CLI, connectors, dashboard, models, etc.) |
| `core/` | Architecture documentation (agent specs, comm protocol, guardrails, WDC panel) |
| `dashboard/` | Prototype dashboard (Python + HTML) |
| `templates/` | Document templates (adversary playbook, evidence intake, etc.) |
| `sync/` | Sync manifest |
| `examples/` | Example domain configs (legal, medical, startup-diligence) |
| `infra/` | Old infrastructure (Docker, Terraform) |
| `config/` | YAML configs (ciphergy.yaml, domain profiles, jurisdictions) |
| `scripts/` | Old utility scripts (agent_comm, asana_bridge, cascade, crypto, etc.) |
| `tests/` | Prototype test files (test_bedrock, test_nerve_center, etc.) |
| `versions/` | Empty version directory |
| `github-workflows/` | Old CI/CD workflow files (deploy.yml, test.yml) |
| `nested-duplicates/` | Duplicate nested directories created by Devin (tests/tests, scripts/scripts, .github/.github) |
| `*.md` | DualAI_README, ENHANCEMENTS, REQUIREMENTS, STANDARD_OPS |
| `setup.py` | Old setuptools config |
| `package.json` | Old Node.js config |
| `*.json` | Old config/registry/alerts JSON files |
| `requirements.txt` | Old pip requirements |
| `launch.sh` | Old launch script |
| `LICENSE` | Original license file |

## Status

Archived on 2026-03-15 during Phase 0 cleanup.
The active codebase lives in `src/`, `tests/`, `infrastructure/`, and `.github/` at the repo root.
